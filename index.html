Library.php

A lean, single-class PHP micro-framework covering database access, routing, view rendering, and common utilities — with strict types, prepared statements, and CSRF protection throughout.

Features

🗄️

Database Layer

Full CRUD with prepared statements, JOINs, pagination, partial search, transactions, and safe identifier quoting.

🔀

HTTP Router

Named route params, route groups, global & per-route middleware, static file serving, and HTMX support.

🔒

Security Built-in

CSRF protection on all mutating methods, bcrypt password hashing, path-traversal-safe static serving, SSL-enforced cURL.

⚡

Zero Dependencies

No Composer packages. Pure PHP 8.1+ with strict types, using only built-in extensions (mysqli, curl, fileinfo).

Quick Start

1

Get an instance

The class uses a singleton — call getInstance() everywhere; it will reuse the same connection.

PHP
$db = Library::getInstance('localhost', 'root', 'secret', 'mydb');
2

Read data

PHP
// All users
$users = $db->readTable('users');

// Paginated, filtered
$admins = $db->readTable(
    table: 'users',
    columns: 'id, name, email',
    limit: 10,
    page: 2,
    where: 'role = ?',
    whereParams: ['admin']
);
3

Register routes

PHP
$db->setViewPath(__DIR__ . '/views');

$db->get('/', fn() => $db->render('home.php'));

$db->get('/users/{id}', function($id) use ($db) {
    $user = $db->readTable('users', where: 'id = ?', whereParams: [$id]);
    $db->sendJSON($user[0] ?? null);
});

$db->run();
4

Write & protect

PHP
// In your form view:
Library::setCsrf(); // emits hidden input

// POST route (CSRF auto-checked by run()):
$db->post('/users', function() use ($db) {
    $id = $db->writeRecord('users', [
        'name'  => $_POST['name'],
        'email' => $_POST['email'],
        'password' => Library::hashPassword($_POST['password']),
    ]);
    Library::redirectWithMessage('/users', 'User created!');
});

group(string $prefix, callable $callback): void
Register a group of routes under a shared URI prefix.
PHP
$db->group('/api/v1', function() use ($db) {
    $db->get('/users', $listUsers);     // → /api/v1/users
    $db->post('/users', $createUser);   // → /api/v1/users
    $db->get('/users/{id}', $getUser);  // → /api/v1/users/{id}
});
public
middleware(callable $middleware, array $routes = []): void
Register a global or route-specific middleware callable.
ℹWhen $routes is empty, the middleware runs on every request. The callable receives ($params, $next) — call $next() to continue the chain.
PHP
// Global auth guard
$db->middleware(function($params, $next) {
    if (!isset($_SESSION['user'])) {
        Library::redirect('/login');
    }
    $next();
});
